# Bike Rental (따릉이) - Fullstack Example

This repository contains a starter fullstack project for a bike-share rental system.

- Backend: Express + MySQL (raw SQL via `mysql2`) (in `server`)
- Frontend: React + Vite (in `client`)

Features included (starter):

- User auth (JWT)
- Admin middleware and admin CRUD endpoints
- Stations API (open/close, status)
- Posts and repair reports endpoints
- Simple React UI with login, stations list, and admin dashboard

See `server/README.md` and `client/README.md` for setup and run instructions.
